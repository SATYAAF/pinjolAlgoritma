Sebelum melakukan peminjaman online seorang harus melengkapi data untuk memenuhi persyaratan.

Data SOP tersebut diantaranya : Nama, umur, alamat, nomor rekening, jumlah pinjaman, jumlah cicilan perbulan, pekerjaan, penghasilan perbulan

Adapun kriteria persyaratan yang harus dipenuhi agar bisa mmelakukan pinjaman adalah :

1. Umur peminjam harus lebih dari 20 tahun
2. Penghasilan peminjam minimal lebih dari 1000000

Kalau persyaratan tidak terpenuhi maka peminjaman di tolak, dan dipersilahkan memenuhi persyaratan kembali
